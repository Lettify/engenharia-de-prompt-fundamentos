# Projeto 3 · Império Natural

Plataforma de cardápio digital para a loja física de produtos naturais
**Império Natural**. As páginas foram estruturadas em HTML, CSS e JavaScript
puros para facilitar o uso local.

## Estrutura

- `index.html` — página inicial com carrossel de destaques e categorias.
- `produtos.html` — lista completa com busca inteligente e visualização em
  lista/grade.
- `produto.html` — página individual de produto com detalhes e alertas.
- `blog.html` — blog com detalhes de ofertas, eventos e anúncios.
- `admin.html` — dashboard administrativo (acesso oculto).

## Acesso administrador

O acesso do administrador não fica visível no menu principal. Para abrir o painel
`admin.html`, use uma das opções ocultas abaixo:

- Segure **Shift** e clique no título “Império Natural” no topo da página.
- Digite a sequência **admin** no teclado (sem foco em campos de formulário).
