# Projeto: Cardápio Digital

## Prompt utilizado para criação
> *Crie uma plataforma que vai servir como cardápio digital de produtos físicos de uma loja física de produtos naturais. O nome da loja é "Império Natural". No site, a página inicial deve ter os destaques e anúncios de ofertas ou eventos passando em um carrossel, e descendo a página o usuário deve conseguir já ter acesso à algumas categorias de seleção de produtos disponíveis na loja. No menu de páginas deve estar listada a página que vai listar todos os produtos disponíveis (com opção de visualização em lista e grade), deve haver uma barra de pesquisa do produto (não usando apenas o nome como referência na palavra-chave, mas também outras informações do produto). Ao clicar em qualquer produto do site, plataforma deve abrir uma página individual do produto onde vão ser disponibilizadas todas as informações que um produto deve ter. Cada produto tem seus tipos de características por categoria, por exemplo os biscoitos - onde deve informar sobre ingredientes para alertar alérgicos, etc. Castanhas, queijos, temperos, etc. Deve haver um acesso administrador oculto para que este tenha acesso a um dashboard de gerenciamento do cardápio, para cadastro de produtos, anúncios de destaques, ofertas e eventos. A plataforma deve ter uma página para servir como algo tipo um blog (para ser usado como detalhes dos destaques de anúncios do carrossel, etc.*
